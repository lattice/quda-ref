00007    into forms in GPU so as to enable coalesce access
00008    The function only converts half(even or odd) of the links

00011

00015      X links                             Y links         T,Z links

00017    The GPU format of data looks like the following
00018    [a0a1][a18a19]  ....[pad][a2a3][a20a21]..... [b0b1][b18b19]....
